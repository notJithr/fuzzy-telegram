## Project Overview

This project is a Ruby on Rails web application hosted on an Azure Linux virtual machine using Infrastructure as a Service. The server is manually configured with Ruby, Rails, Nginx, DNS and SSL/TLS.

The application is a simple vehicle deal management prototype that can record vehicle/customer deal information and display useful status information.
